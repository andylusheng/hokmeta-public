# Attribution

If you use HOKMeta data in a public project, video, spreadsheet, Discord bot, article, or community tool, please credit HOKMeta.

Preferred attribution:

```md
[Data by HOKMeta](https://hokmeta.com)
```

Plain text attribution:

```txt
Data by HOKMeta: https://hokmeta.com
```

Suggested description:

```txt
Hero builds, counters, tier lists, and item data from HOKMeta.
```

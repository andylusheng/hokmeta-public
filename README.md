# HOKMeta

**HOKMeta** is an independent Honor of Kings Global meta database for hero builds, tier lists, counters, items, arcana, patch notes, and ranked climb guides.

Website: https://hokmeta.com

## What Is HOKMeta?

HOKMeta helps Honor of Kings Global players make better ranked decisions with structured game data and practical guides.

The site covers:

- Hero builds
- Tier lists by lane and role
- Counter picks and matchup notes
- Item and arcana references
- Patch and balance tracking
- Ranked climb recommendations
- Beginner and advanced strategy guides

## Use Cases

This project can be useful for:

- Honor of Kings guide writers
- Content creators making build or counter videos
- Community Discord bots
- MOBA data analysis

## Data Coverage

HOKMeta tracks Honor of Kings Global meta data including:

- Hero names, roles, lanes, difficulty
- Win rate, pick rate, ban rate
- Recommended builds
- Arcana and spell setups
- Counter relationships
- Skills and playstyle notes
- Patch history and balance updates
